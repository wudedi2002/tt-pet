# 设计参考图目录

请将改版过程中使用的参考截图放在此文件夹，便于团队在 Cursor 侧栏直接查看。

## 建议放入的文件

| 文件名 | 内容 |
|--------|------|
| `ref-liblib-topbar.png` | Liblib 内容聚合页顶栏 |
| `ref-facemini-sidebar.png` | 原站 Facemini 侧栏 |
| `ref-home-hero.png` | 首页 Hero 参考 |
| `ref-hub-layout.png` | 创作中心布局 |
| `ref-image-studio.png` | 图片创作台 |
| `ref-video-studio.png` | 视频创作台 |
| `ref-logo.png` | 品牌 Logo 原图 |

## 恢复方式

1. 从本地聊天记录 / 截图文件夹找到当时上传的图
2. 复制到本目录并按上表命名
3. 在 Cursor 左侧文件树展开 `design/references/` 即可查看

若你重新上传参考图到对话，我可以帮你整理命名并同步到此目录。
