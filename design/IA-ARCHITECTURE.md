# 信息架构与页面路由

## 页面地图

```
#home                 品牌首页（Landing，无侧栏）
  └─ 开始创作 → #hub

#hub                  创作中心 / 内容聚合
  ├─ 场景卡片 → #image / #video / #avatar / #content
  ├─ 灵感广场
  └─ 常用模板

#image                AI 图片
  ├─ Tab: 图片创作（默认）
  └─ Tab: 我的作品

#video                AI 视频
  ├─ Tab: 视频创作（默认）
  └─ Tab: 历史视频

#avatar               数字人主页面
  └─ 更换形象 → #avatar-select

#avatar-select        数字人形象选择（唯一二级页）
  └─ 选中形象 → 返回 #avatar

#content              爆款图文
  └─ 推送至数字人 → #avatar

#me                   个人中心
  ├─ Tab: 个人中心
  └─ Tab: 我的作品
```

## 侧栏导航（应用内常驻）

| 分组 | 入口 | Hash |
|------|------|------|
| — | 首页 | `#home` |
| — | 创作中心 | `#hub` |
| 创作 | AI 图片 | `#image` |
| 创作 | AI 视频 | `#video` |
| 创作 | 数字人 | `#avatar` |
| 创作 | 爆款图文 | `#content` |
| 我的 | 个人中心 | `#me` |

## 顶栏上下文

| 页面 | 顶栏标题 / Tab |
|------|----------------|
| hub | 创作中心 |
| image | 图片创作 / 我的作品 |
| video | 视频创作 / 历史视频 |
| avatar | 数字人创作 |
| content | 爆款图文 |
| me | 个人中心 / 我的作品 |

## 跨模块联动

1. **爆款图文 → 数字人**：`推送至数字人` 同步文案到 `#avatar`
2. **数字人 ← 图文**：`接收图文文案` 读取草稿
3. **灵感库**：全局侧栏面板，按模块过滤模板
4. **创作中心场景卡片**：直达对应模块

## 仅保留入口（不展开深层页）

- 会员中心 / 账单明细 / 账号设置
- 邀请有礼 / 会员超市
