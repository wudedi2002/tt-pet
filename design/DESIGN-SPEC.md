# Facemini 改版设计规范 v1.0

基于 [facemini.com](https://www.facemini.com/) 第一版迭代改版，沿用黑紫科技风，做视觉规整与组件统一。

## 1. 设计原则

- **保留调性**：深色哑光 + 品牌紫，不更换全新风格
- **简约高级**：去除杂乱装饰，层次分明
- **全局统一**：圆角、按钮、弹窗、卡片、字体层级一致
- **可演示**：所有页面可跳转、可交互，适合企业汇报

## 2. 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景基底 | `#060608` | 页面主背景 |
| 抬升背景 | `#0e0e14` | 分区、输入区 |
| 卡片背景 | `#14141c` | 卡片、面板 |
| 边框 | `#252530` / `#353545` | 分割线、描边 |
| 品牌紫 | `#8b5cf6` | 主按钮、选中态、高亮 |
| 品牌紫 Hover | `#9d6ff8` | 按钮悬停 |
| 正文白 | `#ffffff` | 主文字 |
| 次要文字 | `#a8a8b8` | 说明、副标题 |
| 弱化文字 | `#6e6e80` | 占位、元信息 |
| 告警红 | `#ef4444` | 额度不足、内容违规 |
| 成功绿 | `#22c55e` | 草稿保存、成功提示 |

## 3. 字体与圆角

- **字体**：PingFang SC / Microsoft YaHei / Segoe UI
- **圆角**：`8px`（小） / `12px`（默认） / `16px`（大）
- **标题层级**：H1 36px → H4 15px（见 `css/main.css`）

## 4. 布局结构

### 4.1 首页（`#home`）

- 顶部：胶囊式 Landing 导航（关于我们 / 创作中心 / 登录注册）
- 主体：Hero Banner + 数据指标 + 为什么选择 / 能做什么 + CTA + Footer
- **无左侧栏**

### 4.2 应用内页（`#hub` 及四大模块）

```
┌─────────────┬──────────────────────────────────────────┐
│ 左侧导航     │ 顶栏：折叠 / 面包屑或Tab / 搜索 / 积分 / 头像 │
│ Face mini   ├──────────────────────────────────────────┤
│ 首页        │                                          │
│ 创作中心    │              页面主内容区                  │
│ ─ 创作 ─    │                                          │
│ AI 图片     │                                          │
│ AI 视频     │                                          │
│ 数字人      │                                          │
│ 爆款图文    │                                          │
│ ─ 我的 ─    │                                          │
│ 个人中心    │                                          │
└─────────────┴──────────────────────────────────────────┘
```

顶栏参考 Liblib 内容区风格，侧栏参考原站 Facemini。

## 5. 核心组件

| 组件 | 规范 |
|------|------|
| 主按钮 | 品牌紫底、白字、8px 圆角 |
| 次按钮 | 深灰底 + 边框 |
| 卡片 | `#14141c` 底 + 1px 边框 + 12px 圆角 |
| Tab | 胶囊容器，选中项紫色填充 |
| 输入框 | 深色底，focus 紫色描边 + 外发光 |
| 弹窗 | 四类统一：网络异常 / API 超时 / 额度不足 / 内容违规 |
| Toast | 右上角，成功绿边 / 信息紫边 |
| 生成遮罩 | 全屏 + 进度条 + 场景化状态文案 |

## 6. 模块设计要点

### AI 图片（`#image`）

- 双 Tab：图片创作 / 我的作品
- 创作区：大输入框 + 模型/画幅/数量 + 生成按钮（标注积分）
- 下半区：热门场景灵感库（分类标签 + 卡片流）
- 作品页：全部/收藏/最近7天 + 卡片流（复用/下载/图生图）

### AI 视频（`#video`）

- 双 Tab：视频创作 / 历史视频
- 双模式：图片转视频 / 文案生视频
- 场景模板下拉 + 时长 3s/5s/8s
- 历史：编辑/复用/下载/删除

### 数字人（`#avatar` + `#avatar-select`）

- 三栏：预览区 + 编辑区 + 生成记录
- 预览：横竖屏、无水印、快捷形象、口播模板
- 二级页：职场/主播/卡通/写实分类

### 爆款图文（`#content`）

- 场景模板 + 文案编辑 + 历史记录
- 核心联动：一键推送至数字人

## 7. 全局交互规则

- Hash 路由跳转，刷新保持页面
- 输入区自动草稿（localStorage）
- 生成消耗积分透明展示
- 历史记录支持复用 / 下载 / 删除
- 移动端：侧栏可折叠，标签横向滚动

## 8. 设计基准

- 主设计宽度：**1920px**
- 内容最大宽：1440–1600px
- 响应断点：1200px / 768px
