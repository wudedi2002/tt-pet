# 改版迭代记录

记录 Facemini 原型从初版到当前 Demo 的主要设计变更。

## v0.1 — 初版六页原型

- 顶栏固定导航 + 四大模块入口
- 首页：场景卡片 + 最近使用 + 常用模板
- 各模块：图片双模式、视频双模式、数字人、图文、形象选择

## v0.2 — 创作中心 + 侧栏架构

- 首页精简为 Hero Landing
- 新增 `#hub` 内容聚合页
- 应用内统一左侧导航（Liblib 式工作区）
- Logo 替换为用户提供的紫色六边形

## v0.3 — 顶栏重设计

- 融合 Liblib 内容顶栏 + Facemini 原站侧栏
- 顶栏：折叠按钮 / 页面标识 / 搜索 / 积分 / 通知 / 头像
- 移除重复的顶部模块导航

## v0.4 — 创作台风格（图片 / 视频）

- 参考即梦 / Liblib：居中 Hero + 模型图标行
- 新增灵感库侧栏面板
- 一键做同款横向滚动区

## v0.5 — 图片双 Tab（对齐 Liblib）

- 图片创作 / 我的作品 顶栏 Tab
- 内联生成进度条
- 热门场景灵感库：双层分类标签 + hover 操作按钮

## v0.6 — 视频 / 数字人 / 作品历史

- 视频：历史二次编辑、复用、下载、删除
- 视频卡片 hover 显示编辑（笔）/ 删除（垃圾桶）图标
- 数字人：三栏布局 + 快捷形象 + 口播模板
- 图片/视频「我的作品」对齐卡片流布局

## v0.7 — 当前版本

- 首页 Landing 完整内容区（图6 风格导航）
- 个人中心 `#me` 双 Tab
- 创作中心：Banner 轮播 + 灵感广场 + 常用模板
- 设计规范沉淀至 `design/` 目录
